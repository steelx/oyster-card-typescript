## Setup

```
# install all dependencies.
$ npm i

# Build your app without minification: 
npm run build

# Build your app with minification: 
npm run build.prod

# run unit tests:
npm run test
```

## Run
```

# Start developing and serve your app:
npm start

```
then visit `http://localhost:9000/` to view the code in console.
